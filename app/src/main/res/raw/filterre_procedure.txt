<center>
<p><b>Raccordez une broche sur un fil et l'autre à la terre</b></p>
</center>
-----
<center>
<p><b>Vérification absence de Terre</b></p>
<p>Mettre le Mégohmmètre sur le mode résistance <b>"Ω"</b></p>
<p>Résultat Attendu : Si pas de terre sur le fil, l'appareil doit afficher 100 kΩ</p>
</center>
-----
<center>
<p><b>Distance du fil 1</b></p>
<p>Mettre le Mégohmmètre sur le mode capacité <b>"nF"</b></p>
<p>Résultat Attendu : L'appreil affiche une valeur de capacité correspondant à une distance</p>
</center>
-----
<center>
<p><b>Vérification Distance fil 2</b></p>
<p>Mettre le Mégohmmètre sur le mode capacité <b>"nF"</b></p>
<p>Résultat Attendu : Même distance que pour le fil 1</p>
</center>
-----
<center>
<p><b>Envoi de Tension sur la ligne</b></p>
<p>Mettre le Mégohmmètre sur le mode envoi de tension <b>"MΩ"</b> (50V)</p>
<p>Résultat Attendu : Si la ligne est bonne, ...</p>
</center>
-----
<center>
<p>Fin des mesures, si tout est ok vous pouvez continuer.</p>
